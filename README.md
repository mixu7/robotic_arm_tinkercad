# Robotic Arm - Tinkercad

This project contains the design of a 5-DOF robotic arm created using Tinkercad.

## Files Included:
- `robotic_arm_design.stl` : 3D model of the robotic arm
- `robotic_arm_screenshot.png` : Screenshot of the arm in Tinkercad interface

## Description:
The robotic arm consists of 5 degrees of freedom (DOF) excluding the base. The arm was built using basic geometric shapes in Tinkercad and colored to match common robotic designs. The screenshot was captured to demonstrate ownership and creation of the model.
